Dynamic Photo Gallery

Dynamic Photo Gallery was developed for North Dakota Trade Office as a separate project which was integrated with the company website. The company employees wanted to add more functionality to their website so that they could have more interaction with their customers. When Dynamic Photo Gallery was implemented it provided NDTO customers the ability to view events, exhibitions and business trips organized by NDTO. The Dynamic Photo Gallery consists of administration and public sides. The administration side can be accessed by authorized users only and the public side is available for any website visitor. The administration side provides the functionality to create, update, delete galleries and images. It also allows to add titles and descriptions for galleries and images. The Dynamic Photo Gallery was developed by means of MS Visual Studio 2008, C# programming language and MS SQL Server 2008. You can view THE CODE and SQL QUERIES on my git hub.
The database design of the dynamic gallery can be viewed on this link. The technical documentation about Dynamic Photo Gallery can be downloaded from this link.
The Dynamic Photo Gallery was used by North Dakota Trade Office for 5 years and, since the company redesigned their website, the photo gallery is not used anymore.
