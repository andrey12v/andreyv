Recommendation System in R programming language

The current project was developed for the Artificial Intelligence Class that I took at NDSU. The main requirements of the project were to develop the system with artificial intelligence in it. I developed Recommendation System in R programming language, using User-Based and Item-Based Collaborative Filtering Algorithms. I made different experiments and then compared the accuracy of the recommendation algorithms based on the accuracy of predicted results. In order to get the recommendations we have to go through several steps: we train the system first based on the training data set (CSV file, consisting of thousands of recommendations) => normalize data in order to avoid anomalies (remove uncommon too high or too low ratings) => get the recommendations and then compare them with existing test data in order to know which algorithm provide more accurate results according to different input parameters. The description of the whole project with experiments, calculations and final results can be viewed on THIS LINK and the source code can be viewed on GITHUB.

Moby-Dick Video Game

Moby-Dick game was developed during Fargo Game Development Competition - “Fargo Game Jam”. The competition was organized from July 15 to July 17 in 2016 and was sponsored by Gravity Gaming Company. The main idea of the game is based on the novel about the white whale “Moby-Dick” and Captain Ahab. The whale destroyed Ahab’s ship and severed his leg at the knee. In its turn, the Captain Ahab decided to revenge the whale. The Video Game starts from the scene when the captain Ahab shoots harpoon at the whale. You will need to move captain’s boat from left to right in order to get over the whale and hit button “X” to get the whale on harpoon. As soon as you get the whale on the harpoon you are moved to the next level where you are pulled behind the whale on the big speed and dodge from icebergs by moving to the right and to the left. The executable file for the game can be downloaded HERE. The source code for this video game can be viewed on GITHUB.

Map Reduce in Java

The Map Reduce project was implemented in Java programming language for the class “CSCI 650 - Cloud Computing” that I took at NDSU. The main idea of this project is to calculate weighted sum for every single class (every line) from the input file in the Map function. After that, using the Reduce procedure, group all records together by class number (line number) and calculate average, minimum and maximum values. The project also requires to run Map Reduce on different number of nodes 2, 4, 6, 8, 10, 12, 14 and make analysis of execution time. The requirements for the project can be viewed in this PDF file and the sample input file can be viewed from THIS LINK. The Java code was developed by means on Hadoop library and the source code for this project can be viewed on GITHUB. The whole description of the projects and the final results conclusions can be viewed from THIS LINK.
